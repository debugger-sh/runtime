# runtime

WebAssembly compiler and runtime

## Getting Started

> Requires [Cargo 1.91+](https://crates.io/) and [Node v22+](https://nodejs.org/en). Make sure that `~/.cargo/bin` is added to your `PATH`.

```sh
cargo install wasm-pack
npm install
npm run build
```
